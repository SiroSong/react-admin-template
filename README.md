# react-admin-template

### 简介

react-admin-template 是一个简单、纯净、轻量化的 react 管理后台模版，旨在能够快速简洁的创建基于 react 实现的管理应用。

> | 依赖              | 版本    |
> | ----------------- | ------- |
> | react             | ^18.2.0 |
> | react-dom         | ^18.2.0 |
> | react-router-dom  | ^6.6.1  |
> | antd              | ^5.1.2  |
> | @ant-design/icons | ^4.8.0  |
> | vite              | ^4.0.0  |
> | typescript        | ^4.9.3  |

### 功能

- 路由权限
- 暗黑主题
